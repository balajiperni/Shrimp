About

About This Project

Technical details about the shrimp detection system

Model Architecture

This application uses YOLOv8, the latest generation of the You Only Look Once object detection architecture. YOLOv8 delivers state-of-the-art accuracy while maintaining real-time inference speed, making it ideal for counting tasks in aquaculture and marine research.

YOLOv8 PyTorch Ultralytics

Backend Stack

Built with FastAPI for high-performance async API endpoints. The model runs server-side with GPU acceleration when available, falling back to CPU inference. OpenCV handles image processing and annotation drawing for the output visualizations.

FastAPI OpenCV Python 3.11

Use Cases

Designed for aquaculture farms needing automated shrimp counting, marine researchers studying population dynamics, and quality control in shrimp processing facilities. The system can process batch images for farm-wide monitoring and reporting.
